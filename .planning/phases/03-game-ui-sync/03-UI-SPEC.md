## Interaction Specs

### Card Flip Animation
- Duration: 400ms ease-in-out
- Transform: `rotateY(0 → 180)` via Framer Motion
- Face-down: card back (secondary background + subtle pattern)
- Face-up: icon centered (lucide icon, foreground color, 32px default size)
- Matched: after flip animation completes, add violet border ring + slight scale (1.02) + reduced opacity (90%)
- Mismatch: after 1s delay, flip back with same 400ms animation; both cards animate simultaneously

### Board Layout
- 4 columns × 6 rows responsive grid
- Card aspect ratio: 3:4 (width:height)
- Gap: 8px between cards
- Max board width: 480px (centered)
- Cards scale down proportionally on smaller viewports

### Scoreboard
- Horizontal bar above the board
- Two player slots side by side
- Active player slot has violet left border + bold name
- Each slot: name, score count, move count
- Timer centered between player slots

### Timer
- Display format: "M:SS"
- Updated every second from server `timerTick` events
- Position: center of scoreboard bar

### Winner Modal (Dialog)
- Centered overlay with backdrop blur
- Winner name (or "It's a Draw!") at top in display typography
- Final scores list below
- Elapsed time display
- Two buttons: "Rematch" (primary) + "Leave" (secondary/destructive)
- Confetti burst via canvas-confetti on open (if winner, not draw)
- Close on Escape key

### Loading State
- Skeleton shimmer on 24 card positions in 4×6 grid
- Skeleton uses muted background with animated pulse
- Duration until `gameStart` event fires

### Turn Flow
1. Player A's turn: "Your Turn" on A's side, cards clickable for A
2. Player A clicks card 1 → 400ms flip → icon revealed
3. Player A clicks card 2 → 400ms flip → icon revealed
4a. Match: "Match!" toast → both cards get matched styling → A plays again
4b. Mismatch: board locks → 1s wait → both cards flip back → turn switches
5. Player B's turn: "Your Turn" on B's side, cards clickable for B
